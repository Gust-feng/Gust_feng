# CPU的组成

**CPU由控制器、寄存器、时钟、运算器组成**

- 控制器：负责将内存中的数据和指令存入寄存器
- 寄存器：用于存放指令，数据这些操作对象的空间
- 时钟：负责CPU工作节律的时钟信号
- 运算器：负责运算从内存中读入寄存器中的数据

# 代码的翻译

**编译型语言：**

​	在RAM中存储对应的机器码，PC（%程序计数器）指针直接指向代码逻辑

**解释型语言：**

​	在RAM中存储解释器的机器码和源码文件，CPU始终在运行解释器语言，解释器一直在解读源码（JIT可能临时生成一段机器码）

> 解释型语言在运行效率上常低于编译型语言

# 程序运行

程序运行时，OS将磁盘中数据复制进RAM，运行数据包括：经过翻译后的二进制指令、变量和不同资源、堆栈。











# 函数调用的原理

不同于顺序，条件分支和循环，函数调用后需要返回被调用时的地方（即函数调用指令的下一个指令）继续执行，因此函数调用需要使用调用指令和返回指令；

调用指令会在函数入口地址设置到程序计数器前将函数调用的下一条指令的地址保存到栈的内存空间中；

在函数体执行完后，将会执行返回指令

返回指令将保存在栈中的地址设置到程序计数器中。
